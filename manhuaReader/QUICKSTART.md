# 快速开始指南 (QUICKSTART)

## 🚀 5 分钟快速上手

### 第一步：安装 Tampermonkey

1. **Chrome/Edge 用户**
   - 访问 [Chrome 网上应用店](https://chrome.google.com/webstore/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo)
   - 点击"添加至 Chrome"
   - 确认安装

2. **Firefox 用户**
   - 访问 [Firefox 附加组件](https://addons.mozilla.org/firefox/addon/tampermonkey/)
   - 点击"添加到 Firefox"
   - 确认安装

### 第二步：安装漫画阅读器脚本

**方法一：直接安装（推荐）**
1. 点击链接：[manhuaReader.user.js](manhuaReader.user.js)
2. Tampermonkey 会自动打开并显示安装界面
3. 点击"安装"按钮

**方法二：手动安装**
1. 打开 Tampermonkey 管理面板
2. 点击"+"创建新脚本
3. 复制 `manhuaReader.user.js` 的全部内容
4. 粘贴到编辑器
5. 按 `Ctrl+S` 保存

### 第三步：访问漫画网站

目前支持的网站：
- **再漫画**：https://manhua.zaimanhua.com/view/* （✨ SPA 支持）
- **漫画柜**：https://www.manhuagui.com/comic/*/*.html

访问任意漫画章节页面，脚本会自动加载阅读器。

> **提示**：
> - 再漫画是单页应用（SPA），章节切换无需刷新页面，自动加载新数据
> - 漫画柜网站首次访问时会缓存章节列表（1小时），后续访问速度更快！

## 📖 基本使用

### 阅读漫画

#### 翻页操作
```
┌─────────────┬──────────────┬─────────────┐
│             │              │             │
│   点击左侧   │  点击中间     │  点击右侧    │
│   上一页     │  显隐工具栏   │  下一页      │
│             │              │             │
└─────────────┴──────────────┴─────────────┘
```

#### 键盘快捷键
- `←` 左箭头：上一页
- `→` 右箭头：下一页
- `↑` 上箭头：上一章
- `↓` 下箭头：下一章
- `Space` 空格：显示/隐藏 UI
- `Enter` 回车：确认对话框或切换 UI
- `Esc`：关闭 UI 元素（优先级：对话框 > 设置面板 > 侧边栏 > 工具栏）

> **提示**：阅读器采用智能事件拦截技术，**自动放行功能键（F1-F12）和组合键（Ctrl/Shift/Alt）**，确保不影响浏览器快捷键和开发调试。快捷键仅在阅读器可见时生效。

### 使用侧边栏

1. **打开侧边栏**
   - 点击工具栏左侧的 ▶ 按钮
   - 或首次进入时自动显示

2. **侧边栏功能**
   - 查看漫画标题、作者、状态和标签
   - 查看当前章节信息和页数
   - 点击"上一章"/"下一章"按钮跳转
   - 浏览完整章节列表，点击任意章节跳转

3. **关闭侧边栏**
   - 再次点击 ◀ 按钮
   - 或按 `Esc` 键
   - 或点击中间区域隐藏工具栏

### 阅读器设置面板

点击工具栏的设置按钮（⚙️）可以打开阅读器设置面板：

#### 主题风格设置
- **亮色**：浅色背景，适合白天使用
- **暗色**：深色背景，适合夜间使用
- 选择后立即生效并自动保存

#### 预载数量设置
- **范围**：0-5 张图片
- **功能**：控制当前页前后各预加载的图片数量
- **建议**：
  - 网络较好：设置为 2-3，提升翻页流畅度
  - 节省流量：设置为 0-1，仅加载当前页
  - 平衡模式：设置为 2（默认值）
- 设置后立即生效并自动保存

#### 关闭设置面板
- 点击右上角 × 按钮
- 或点击遮罩层空白处
- 或按 `Esc` 键

### 切换主题

#### 方式一：通过设置面板（推荐）
1. 点击工具栏的设置按钮（⚙️）
2. 在"主题风格"选项中选择"亮色"或"暗色"
3. 选择后立即生效并自动保存

#### 方式二：编程方式
```javascript
// 在控制台执行
$vmr.$vm.setupState.toggleTheme()  // 切换主题
```

### 持久化存储
- 使用 `GM_setValue` 保存用户偏好（主题、预载数量）
- 下次打开时自动应用上次选择的设置
- 默认主题为亮色（light），默认预载数量为 2

## ✨ SPA 特性（再漫画专用）

### 什么是 SPA？

单页应用（Single Page Application）在章节切换时不会刷新整个页面，而是通过 JavaScript 动态更新内容。

### SPA 优势

**传统网站：**
1. 点击"下一章" → 页面刷新 → 等待加载 → 显示新章节
2. 每次切换都需要完整的 HTTP 请求

**SPA 网站（再漫画）：**
1. 点击"下一章" → URL 变化 → 自动加载数据 → 平滑过渡
2. 无需页面刷新，体验更流畅

### 自动路由检测

- ✅ 章节切换时自动检测 URL 变化
- ✅ 自动加载新章节数据
- ✅ 离开阅读页时自动隐藏阅读器
- ✅ 返回阅读页时自动显示阅读器

### 使用示例

**场景 1：连续阅读**
1. 访问 `/view/123` → 阅读器自动加载第 123 话
2. 点击"下一章" → URL 变为 `/view/124`
3. 阅读器自动更新为第 124 话（无刷新）
4. 继续点击"下一章" → 无缝切换

**场景 2：离开和返回**
1. 在阅读页 `/view/123` → 阅读器显示
2. 点击漫画标题跳转到详情页 `/details/xxx`
3. 阅读器自动隐藏，显示入口按钮
4. 点击其他章节回到阅读页 → 阅读器重新显示

## 💡 实用技巧

### 技巧 1：快速跳转章节

**方法一：使用侧边栏**
1. 打开侧边栏（点击 ▶）
2. 在章节列表中找到目标章节
3. 点击该章节即可跳转

**方法二：键盘操作**
- 在最后一页按 `→`：跳转到下一章（需确认）
- 在第一页按 `←`：跳转到上一章（需确认）

**方法三：SPA 站点直接点击**
- 在再漫画等 SPA 站点，直接点击原网站的"下一章"按钮
- 阅读器会自动检测并加载新数据

### 技巧 2：最大化阅读空间

1. 按 `Esc` 隐藏所有 UI
2. 只保留漫画图片
3. 需要时再点击中间区域显示工具栏

### 技巧 3：防止误操作

- 首次进入时，点击区域会被锁定 1.5 秒
- 章节边界跳转会显示确认对话框
- 可以点击"取消"放弃跳转

### 技巧 4：自定义主题

在浏览器控制台执行：

```javascript
// 修改背景色为深蓝色
const container = document.querySelector('.manga-reader-container')
container.style.setProperty('--vmr-bg-primary', '#1a237e')
container.style.setProperty('--vmr-bg-secondary', '#283593')

// 恢复默认
location.reload()
```

## 🔧 常见问题

### Q1: 脚本没有自动加载？

**解决方法：**
1. 检查 Tampermonkey 是否启用
2. 检查脚本是否启用（绿色开关）
3. 刷新页面重试
4. 查看浏览器控制台是否有错误信息

### Q2: 如何手动加载数据？

在浏览器控制台执行：

```javascript
$vmr.setMangaData({
  manga: {
    title: "测试漫画",
    author: "测试作者"
  },
  chapter: {
    current: {
      id: "1",
      name: "第1话",
      url: "/chapter/1",
      images: ["image1.jpg", "image2.jpg"]
    },
    previous: null,
    next: null,
    list: []
  }
})
```

### Q3: 如何恢复到默认状态？

**方法一：刷新页面**
- 直接按 `F5` 或 `Ctrl+R`

**方法二：清除主题设置**
```javascript
GM_setValue('vmr-theme', 'light')
location.reload()
```

### Q4: 支持哪些网站？

目前支持：
- **再漫画** (zaimanhua.com) - ✨ **SPA 支持**，章节切换无需刷新
- **漫画柜** (manhuagui.com) - 支持智能缓存，加载更快

如需支持其他网站，请参考 [README.md](README.md) 中的"添加新网站适配器"章节。

### Q5: 如何自定义预载数量？

**方法一：通过设置面板（推荐）**
1. 点击工具栏的设置按钮（⚙️）
2. 在"预载数量"选项中选择 0-5
3. 设置立即生效并自动保存

**方法二：编程方式**
```javascript
// 在控制台执行
$vmr.$vm.setupState.preloadCount.value = 3  // 设置为3
```

**预载数量说明：**
- **0**：不预载，仅加载当前页（最省流量）
- **1-2**：轻度预载，适合一般网络环境
- **3-5**：重度预载，适合高速网络（最流畅）

### Q6: 缓存是如何工作的？

**漫画柜网站的缓存机制：**
- **首次访问**：从详情页提取章节列表并缓存（1小时）
- **后续访问**：直接使用缓存，无需再次请求详情页
- **自动清理**：脚本启动时自动删除过期缓存
- **手动清除**：在 Tampermonkey 设置中清除脚本数据即可

**优势：**
- ⚡ 加载速度提升 80%+
- 🌐 减少网络请求
- 💾 节省流量

### Q7: SPA 站点为什么能自动切换章节？

**工作原理：**
1. 脚本每 500ms 检测一次 URL 的 pathname 部分
2. 检测到变化后，判断新页面是否为阅读页
3. 如果是阅读页，自动提取并加载新数据
4. 如果不是阅读页，隐藏阅读器

**性能影响：**
- CPU 占用：< 0.1%
- 内存占用：约 1KB
- 对用户完全透明，无感知
