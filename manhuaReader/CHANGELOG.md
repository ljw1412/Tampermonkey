### 🔧 技术改进
- **响应式状态管理**：

- **计算属性优化**：
  - `preloadImages` 根据 `preloadCount` 动态计算预加载图片

## [2.1.0] - 2026-05-18

### ✨ 新增功能
- **SPA 路由监听**：完美支持单页应用（SPA），自动检测 pathname 变化并重新加载数据
  - 使用 `setInterval` 定时检测路由变化
  - 仅在标记为 `spa: true` 的站点启用
  - 可配置的轮询间隔（`pathnamePollingDelay`）
  - 离开阅读页时自动隐藏阅读器
- **增强日志系统**：完善的日志输出，便于调试和问题排查
  - 网站检测日志：显示查找过程和匹配结果
  - 页面类型检查日志：显示路径匹配状态
  - 数据加载日志：显示提取进度和结果
  - 路由变化日志：显示 pathname 变化详情
- **配置化适配器**：每个站点可独立配置行为参数
  - `spa`: 是否启用路由监听
  - `loadDelay`: 数据加载延迟时间
  - `pathnamePollingDelay`: SPA 路由检测间隔
- **统一加载策略**：`loadData()` 函数统一管理初始化和路由变化的数据加载
  - 自动检测页面类型
  - 根据配置延迟加载数据
  - 控制入口按钮显示状态

### 🔧 技术改进
- **数据加载器重构**：
  - `getWebsite()`: 独立的网站检测函数，返回匹配的站点配置
  - `checkReadPage()`: 检查当前页面是否为阅读页
  - `loadMangaData()`: 异步加载漫画数据，支持跳过重复检查
  - `loadData()`: 统一的加载入口，用于初始化和路由变化
- **SPA 路由监听模块**：
  - `startPathnameTimer()`: 启动路由监听，支持自定义回调和间隔
  - `stopPathnameTimer()`: 停止路由监听，防止内存泄漏
  - 智能防抖：避免短时间内多次触发
- **全局 API 优化**：
  - `$vmr.setEntryVisible()`: 控制阅读器入口按钮显示
  - `$vmr.setReaderVisible()`: 控制阅读器界面显示/隐藏
  - `$vmr.setMangaData()`: 设置漫画数据
  - 修复 Vue 3 CDN 版本的实例访问方式（`window.$vm = $vm._`）
- **错误处理增强**：
  - 非阅读页时抛出明确错误信息
  - 数据提取失败时记录详细错误日志
  - SPA 路由变化时的异常捕获

### 🎨 UI/UX 优化
- **SPA 无缝体验**：章节切换无需刷新页面，数据自动更新
- **智能显示控制**：
  - 进入阅读页：自动显示阅读器
  - 离开阅读页：自动隐藏阅读器，显示入口按钮

- **日志可视化**：清晰的日志前缀和结构化输出

### 📝 代码质量

## 版本说明

### 版本号规则

采用语义化版本 2.0.0 格式：**主版本号.次版本号.修订号**

- **主版本号**：不兼容的 API 修改
- **次版本号**：向下兼容的功能性新增
- **修订号**：向下兼容的问题修正

### 类型说明

- **✨ 新增功能 (Added)**：新功能
- **🔄 变更 (Changed)**：现有功能的变更
- **🐛 修复 (Fixed)**：Bug 修复
- **⚠️ 废弃 (Deprecated)**：即将移除的功能
- **❌ 删除 (Removed)**：已移除的功能
- **🔒 安全 (Security)**：安全相关的修复
